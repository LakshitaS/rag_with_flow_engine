Connectors for watsonx.ai

You can add connections to a broad array of data sources for working with data in watsonx.ai. Source connections can be used to read data; target connections can be used to load (save) data. When you create a target connection, be sure to use credentials that have Write permission or you won't be able to save data to the target.

From a project, you must create a connection to a data source before you can read data from it or load data to it.
